PhD Candidate CV Analyzer: What Research Experiences Lead to Top Economics Programs?

February 16, 2026
TL;DR I built a system to analyze CVs from PhD candidates at 15 top economics programs to identify patterns in research backgrounds, predoc experiences, and institutional pathways. Key finding: approximately 67% of students had predoc research experience before entering their PhD programs, with Federal Reserve Board, NBER, and top business schools being the most common predoc institutions.

As an undergraduate interested in pursuing a PhD in economics, I've always wondered: what specific research experiences maximize the probability of admission to top programs? To answer this question, I built the PhD Candidate CV Analyzer—a comprehensive system that scrapes, analyzes, and visualizes CV data from students at 15 top economics PhD programs.

The System

The analyzer consists of five main components:

  • Data Collection: Custom scrapers for each of the 15 target universities (MIT, Harvard, Stanford, Princeton, Chicago, UC Berkeley, Yale, Columbia, UPenn, Northwestern, LSE, Oxford, Cambridge, NYU, UCLA)
  • Data Storage: SQLite database with structured student records
  • Data Export: CSV and JSON export functionality for external analysis
  • Statistical Analysis: Descriptive statistics and pathway analysis
  • Interactive Dashboard: Streamlit-based visualization tool with real-time filtering

The system extracts key information from each CV: undergraduate institution, predoc research experience (if any), predoc institution, PhD start year, and current program. All data collection respects robots.txt and includes rate limiting to be respectful to university servers.

Implications

These findings have several implications for prospective PhD students:

  • Predoc positions matter: With 67% of students having predoc experience, these positions appear to be a significant factor in admissions to top programs.
  • Institutional pathways: Certain undergraduate institutions have strong track records of placing students into top PhD programs, suggesting that institutional reputation and research opportunities matter.
  • Research experience timing: The prevalence of predoc experience suggests that research experience after undergraduate studies (rather than just during) is increasingly important.

Technical Details

The system is built with Python and uses several key technologies:

  • Scraping: Playwright for dynamic content, httpx for API calls, pdfplumber for CV extraction
  • Data Management: SQLModel for ORM, SQLite for storage, Pydantic for validation
  • Visualization: Streamlit for the dashboard, Plotly for interactive charts
  • Analysis: Pandas for data manipulation, NumPy for statistical analysis

The codebase follows a modular architecture with separate layers for collection, storage, analysis, and visualization. This makes it easy to extend the system with new universities, additional analysis, or different visualization approaches.

Future Work

There are several directions for future research:

  • Publication analysis: Extract and analyze publication patterns to see if certain types of publications correlate with admission
  • Fellowship tracking: Identify fellowship programs (NSF, Fulbright, etc.) that correlate with top-tier admission
  • Institution normalization: Better handling of institution name variations to improve feeder analysis
  • Machine learning models: Build predictive models to identify factors that maximize admission probability

Code and Data

The complete codebase is available on GitHub: github.com/mwballif/phd-cv-analyzer. The repository includes:

  • All 15 university scrapers
  • Database schema and models
  • Statistical analysis tools
  • Interactive dashboard code
  • Comprehensive documentation

All data collection respects robots.txt and includes appropriate rate limiting. The system is designed to be ethical and respectful of university websites while providing valuable insights for the research community.

Conclusion

This project demonstrates how systematic data collection and analysis can reveal patterns in academic pathways that might not be immediately obvious. While correlation doesn't imply causation, the data suggests that predoc research experience has become a significant factor in admissions to top economics PhD programs. For prospective students, this highlights the importance of seeking out research opportunities, whether through predoc positions, undergraduate research, or other pathways.

The system is designed to be extensible—new universities can be added, additional analysis can be performed, and the dashboard can be customized for different research questions. I hope this tool proves useful for prospective PhD students, academic advisors, and researchers studying academic pathways.
